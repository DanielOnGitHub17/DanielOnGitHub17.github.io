This Python tkinter application navigates through a file system easily.
It uses various tkinter widgets, Canvas, Button, Label, OptionMenu, Checkbutton, ScrollBar, to provide a nice interface.

It also uses standard library modules, os, tkinter (ttk, messagebox, simpledialog...), pathlib, datetime
to provide the functionality of the application

Xplorer can perform creating, renaming, sorting, starting, archiving, deleting, and showing properties of files and folders.
